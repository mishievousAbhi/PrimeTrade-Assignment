# PrimeTrade.ai Assignment

## Objective
Analyze the relationship between Bitcoin Fear & Greed Index and Hyperliquid trader performance.

## Project Structure
- PrimeTrade_Assignment.ipynb
- historical_data.csv
- fear_greed_index.csv
- README.md

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Analysis Performed
- Data Cleaning
- Data Merging
- Exploratory Data Analysis (EDA)
- Sentiment-wise Performance Analysis
- Visualizations
- Key Insights

## How to Run
1. Clone the repository.
2. Install the required libraries.
3. Open the notebook in Jupyter Lab.
4. Run all cells.
